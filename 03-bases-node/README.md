# Notas:
Programa que guarda e imprime una talb de multiplicar
```
Opciones:
      --help     Muestra ayuda                                        [booleano]
      --version  Muestra número de versión                            [booleano]
  -b, --base     Base para tabla de multiplicar             [número] [requerido]
  -h, --hasta    Hasta que numero llegara la tabla de multiplicar
                                                          [número] [defecto: 10]
  -l, --listar   Muestra la tabla en consola         [booleano] [defecto: false]
```
